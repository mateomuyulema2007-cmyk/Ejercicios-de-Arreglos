# Ejercicios-de-Arreglos
Sistema de Gestión y Análisis de Calificaciones
Este programa automatizado permite procesar las notas del examen final de un grupo de estudiantes, generar reportes estadísticos y realizar búsquedas rápidas. Cuenta con implementaciones independientes en C++ y Java.
Requerimientos
Registrar Calificaciones: Ingreso de nombres y notas de los alumnos.
Reporte Estadístico: 
Muestra:El promedio general del grupo.
Total de aprobados (nota >= 7).
Total de reprobados (nota <= 7).
Buscar Estudiante: Búsqueda secuencial por nombre.
Si existe: Muestra nombre, nota y si está "Aprobado" o "Reprobado".
Si no existe: Muestra un mensaje de error.
Tecnologías
C++ (Estructuras de control y arreglos)
Java (Consola con Scanner y arreglos)
